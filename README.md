# 轱辘 - 一个 Vue UI 组件库

作者 瓜坤

## 安装

使用本框架前，请在 CSS 中开启 border-box

```
* {box-sizing: border-box;}
```

```
    :root {
      --button-height: 32px;
      --button-bg: white;
      --button-active-bg: #eee;
      --font-size: 14px;
      --border-radius: 4px;
      --color: #999;
      --border-color: #999;
      --border-color-hover: #666;
    }
```
